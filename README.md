# vscode-hot-key

```
ctrl + shift + p 命令區

f2 變更圈選變數

ctrl + D 選取文字統一修改

```
## HTML

```

.name     新增一個 class=name 的 div

!         新增一個 html 模板

link:css  新增 css link 連結

script:src 新增 script src 

ctrl + /  註解

tagName * n  新增 n 個 tag

ul>li*3    連接父子元素

div+ul     連接同層元素

tag.class  tag 加上 class 

tag#id     tag 加上 id

tag{content} tag 增加 content

tag[attr] 為 tag 新增 attr
```
